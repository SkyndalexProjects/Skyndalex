# Running example

```
npm install
npm run build
npm run dev
```

# Running on Docker

## From Preview

```
sudo docker build --tag skyndalex:preview .
sudo docker compose -f docker-compose:preview.yml up
```

## From Stable

```
sudo docker build --tag skyndalex:stable .
sudo docker compose up
```
